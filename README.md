# ClickLight

Highlights your clicks during live demos, screen sharing, and presentations. Originally by [@aurorascharff](https://github.com/aurorascharff/ClickLight) for macOS — this fork adds a **Windows port**.

## Windows

Download `ClickLight-Windows-x64.zip` from [Releases](https://github.com/pvjagtap/ClickLight/releases), extract, and run. No installation needed.

See [ClickLight.Windows/README.md](ClickLight.Windows/README.md) for build instructions.

## macOS

Install with Homebrew from the [upstream repo](https://github.com/aurorascharff/ClickLight):

```bash
brew tap aurorascharff/clicklight https://github.com/aurorascharff/ClickLight
brew install --cask aurorascharff/clicklight/clicklight
```

## Features

- Click highlights across all apps
- Separate visuals for press, release, right-click, and drag rectangle
- Settings window with sliders for size, duration, intensity, and color
- System tray (Windows) / menu bar (macOS) with quick presets
- Custom color picker
- Launch at Login
- Multi-monitor support
